apsth

Spike density function with adaptive Gaussian kernel.

Syntax

[psth_aconv psth_sd] = apsth(binraster,dt,time)

Description

[psth_aconv psth_sd] = apsth(binraster,dt,time) computes the spike density function with an adaptive Gaussian kernel optimized for local probability of spiking. For near-zero spiking probability, the kernel is flat resulting in a moving average. For a probability of 1, the kernel converges to a Dirac delta. Note that the mapping of probabilities to kernel width is arbitrary between the extremes. In this implementation, the alpha parameter of the gausswin function (which is inversly related to the sd of the Gaussian window) is linearly mapped to probabilities.
