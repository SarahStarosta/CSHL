addcell

Add a cell to the database and perform all the analyses.

Syntax

ok = addcell(cellid)
ok = addcell(cellid,'quiet')

Description

ok = addcell(cellid) adds the specified cell to the database and performs the analyses. cellid may also be a cell array of strings. addcell returns 1 if no errors occur and 0 otherwise.

ok = addcell(cellid,'quiet') suppresses text displays.
