set_subplots

Make subplots with defined distances.

Syntax

h = set_subplots(nrows,ncols,interw,interh,param,value)

Description

h = set_subplots(nrows,ncols,interw,interh,param,value) creates subplots in an nrows x ncols arrangement with horizontal and vertical distances between the plots given in interw and interh, respectively. The parameter, value pairs can be used to specify axes properties similar to subplot.
