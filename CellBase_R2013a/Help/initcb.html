initcb

Initialize CellBase.

Description

initcb specifies default directory and main database file for CellBase. It also set CellBase preferences.

initcb(name) creates new CellBase with the specified name. Multiple instances of CellBase can be initialized with different names. Default CellBase can be set using choosecb.

See Also

loadcb, choosecb, deletecb, addnewcells
