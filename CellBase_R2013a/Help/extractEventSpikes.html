extractEventSpikes

Extract and align spikes in each trial relative to trial events.

Syntax

[event_stimes,event_windows] = extractEventSpikes(cellid,events)

Description

Input arguments:

cellid - ID of a unit in CellBase

events - N x 4 cell array of events (see defineEventsEpochs)

Output arguments:

event_stimes - 1 x N cell array of 1 x TrialNum cell arrays of spike times relative to the first event trigger in 'events'.

event_window - 1 x N cell array of 1 x TrialNum arrays noting the time window of extraction for each trial.
