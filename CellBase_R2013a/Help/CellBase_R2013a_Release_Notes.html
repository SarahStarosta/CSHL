Release Notes

Contents

  • New features
  • Changes
  • Bug fixes

New features

  • new database, data preprocessing and analysis functions
  • new functions include setvalue, filterTrials, ultimate_psth, psth_stats, a set of functions for optical tagging analysis and many more
  • complete documentation
  • help headers, comments, code cleanup
  • input argument handling with inputParser for most of the complex functions
  • new directory structure for .m files
  • insertdata, prealignSpikes, selectcell, addanalysis, addcell - added functionalities
  • most of the functions updating CellBase save a time-stamped backup (like addanalysis, delanalysis, etc)

Changes

  • TheMatrix has been converted to a cell; it is kept compatible with previous versions (hopefully)
  • Time stamp conversion is handled by a new preference
  • getvalue returns values in the order of the cell IDs
  • output arguments of findanalysis and findprop were changed

Bug fixes

  • addcell: handling analyses when adding a new cell has been fixed
  • partition_trials: NaNs are not considered a separate partition anymore
  • MakeTrialEvents2_gonogo: TTLs parsed according to their channels
  • findSegs2 contained many bugs; findSegs3 added, with hopefully most of the bugs eliminated; findSegs3 renders findSegs2 obsolete
  • addanalysis: output arguments of an added analysis do not have to be wrapped in a cell array

If you would like to find out more, please consult CellBase editlog.
