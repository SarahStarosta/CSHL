binraster2dapsth

'Doubly adaptive' PSTH from binned spike raster.

Syntax

[psth, spsth, spsth_se] = binraster2dapsth(binraster,dt,sigma,comp,valid_trials)

Description

[psth, spsth, spsth_se] = binraster2dapsth(binraster,dt,sigma,comp,valid_trials) calculates 'doubly adaptive' peri-event stimulus histogram (psth; see dapsth), smoothed PSTH (spsth) and standard error of smoothed PSTH (spsth_se) from binraster at time resolution dt. The output is restricted to valid_trials (1 x num_trials). comp: M x num_trials matrix of integers (0...n_condition), the matrix for partitioning trials. sigma determines the smoothing kernel for the smoothed PSTH (see smoothed_psth).

See Also
