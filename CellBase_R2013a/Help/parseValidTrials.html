parseValidTrials

Trial selection.

Syntax

valid_trials = parseValidTrials(ve,event,valid_trials)

Description

parseValidTrials is a helper function for input argument handling. valid_trials = parseValidTrials(ve,event,valid_trials) checks for NaNs in the field corresponding to the event in the events structure (ve, stimulus or trial events). It handles valid_trials input in both logical and index set format.
