selectcell

Find cells satisfying certain criteria.

Syntax

cellids = selectcell(selstr)

Description

cellids = selectcell(selstr) executes the "logical expression" in selstr to find cell IDs matching a set of criteria.

The "logical expression" is constructed of criteria based on the values of particular properties (type listtag('property') for a list of all properties in CellBase) connected with logical operators & (and) and | (or). For example '"Rate" > 5 | "Selectivity" > 0.2' will return cell IDs whose "Rate" is either greater than 5 or "Selectivity" is greater than 0.2. "Rate" and "Selectivity" must be existing properties in CellBase (they can be added using addanalysis or insertdata).

Operations including string properties are also supported, as in the following example:
selstr = ['ismember("Area1",{''GP'',''GP/SI'',''SI'',''IC'',''RT/IC'',''EP'',''EA'',''EAC''})&"PV+"==1&"ID_PC">20'];
PV = selectcell(selstr);
