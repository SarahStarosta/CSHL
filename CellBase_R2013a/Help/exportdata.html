exportdata

Export all data introduced to CellBase by the 'insertdata' function.

Syntax

data = exportdata('var')

Description

exportdata saves all variables from 'TheMatrix' which corresponds to 'insertdata' in analyses to a cell table under the name 'CellBase_InsertData' in the CellBase directory (see CellBase documentation).

exportdata(filename) exports the data to a file named filename in the current directory.

data = exportdata('var') exports the data to a variable instead of saving.
