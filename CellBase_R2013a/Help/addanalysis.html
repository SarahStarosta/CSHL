addanalysis

Add an analysis to CellBase.

Description

addanalysis(funhandle) adds an analysis to CellBase with the specified function handle. addanalysis also executes the analysis function passed by funhandle on the cells included in CellBase. Note that the analysis specified by funhandle should take a cell ID as its first input argument.

addanalysis(funhandle,'property_names',pr) passes the list of property names (pr, cell array of strings) to addanalysis. The property name 'default' is used if no property names are specified. The number of property names listed should match the number of outputs of the analysis function.

addanalysis(funhandle,'property_names',pr,'arglist',argval) also passes input argument-value pairs (argval, N-by-2 cell array) to addanalysis. These will be passed on to the analysis function.

Examples

addanalysis(@LRatio2,'property_names',{'ID_PC','Lr_PC'},...
    'arglist',{'feature_names' {'WavePC1' 'Energy'}}) ;
