iscellid

Check if CellBase cell ID exists.

Syntax

yn = iscellid(cellid)

Description

yn = iscellid(cellid) returns true if the referred cell ID exists and false if it doesn't. It checks the existance of a .mat file associated with the cell (see validcellid) and checks whether the cell ID was stored as a new cell ID.
