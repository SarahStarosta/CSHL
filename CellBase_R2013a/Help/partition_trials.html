partition_trials

Sort all trials to partitions.

Syntax

[trials, tags, rtags, num_tags, partnum] = partition_trials(te,partitions)
[trials, tags, rtags, num_tags, partnum] = partition_trials(te,partitions,validtrials)

Description

[trials, tags, rtags, num_tags, partnum] = partition_trials(te,partitions) sorts all trials in te structure to nonoverlapping groups according to partitions. Partiotioned trials (trials) are returned with corresponding tags (tags, rtags) and number of elements in each partition (partnum).

[trials, tags, rtags, num_tags, partnum] = partition_trials(te,partitions,validtrials) restricts the partitioning to valid trials.
