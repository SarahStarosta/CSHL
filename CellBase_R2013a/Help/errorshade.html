errorshade

Shaded error bar plot.

Syntax

h = errorshade(x,y,sd,linecolor);
h = errorshade(x,y,sd,'linecolor',lclr,'linewidth',lw,'shadecolor',sclr);

Description

h = errorshade(x,y,sd,linecolor) plots y against x in LineColor. A shaded error bar is added between y-sd and y+sd. Output returns the handles for the line and the shaded error bar. If sd is a 2-by-N matrix, the two rows are used as upper and lower bound of the shade.

h = errorshade(x,y,sd,'linecolor',lclr,'linewidth',lw,'shadecolor',sclr) - optional arguments LineWidth and ShadeColor default to 2 and grey. LineColor defaults to blue - you can use shortend styles as well ('r:').
