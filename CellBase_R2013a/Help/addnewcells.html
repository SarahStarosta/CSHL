addnewcells

Add new cells to CellBase.

Syntax

nm = addnewcells
nm = addnewcells('prealign',true)
nm = addnewcells('quiet',true)
nm = addnewcells('dir',dr)

Description

nm = addnewcells adds new cells found in Cellbase directory structure to CellBase calling findallcells. It calls addcell, which also performs all the analyses. Number of newly added cells is returned.

nm = addnewcells('prealign',true) calls prealignSpikes.

nm = addnewcells('quiet',true) suppresses message displays.

nm = addnewcells('dir',dr) looks for new cells in dr. dr should contain the path downstream of the CellBase directory, e.g.
addnewcells('dir','n046\130101a\')

Examples

nm = addnewcells('dir','n046\130101a\')

See Also

findallcells, addcell
