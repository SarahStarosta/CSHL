findcellpos

Locate animal, session or tetrode positions in CellBase.

Syntax

pos = findcellpos(cellid)
pos = findcellpos('rat',ratid)
pos = findcellpos('rat',ratid,'session',sessionid)
pos = findcellpos('rat',ratid,'session',sessionid,'tetrode',tet)

Description

pos = findcellpos(cellid) returns the position of cellid in CELLIDLIST.

pos = findcellpos('rat',ratid) locates an animal in CELLIDLIST. 'animal' can be used as an alternative for 'rat' in all sytaxes.

pos = findcellpos('rat',ratid,'session',sessionid) locates a session in CELLIDLIST.

pos = findcellpos('rat',ratid,'session',sessionid,'tetrode',tet) locates a tetrode in CELLIDLIST.

findcellpos returns the matching position indices in CellBase for a particular animal, session, cell or tetrode.
