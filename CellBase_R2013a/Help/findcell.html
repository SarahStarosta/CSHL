findcell

Locate animals, sessions, or tetrodes in CellBase.

Syntax

cellids = findcell('rat',ratid)
cellids = findcell('rat',ratid,'session',sessionid)
cellids = findcell('rat',ratid,'session',sessionid,'tetrode',tetrode)

Description

findcell returns the cell ID(s) in CellBase for a particular rat, session or tetrode.

See Also

addnewcells, addcell, findcellstr, findallcells
