valuecrossing

Interpolate value-crossings of a function.

Syntax

t = valuecrossing(x,y,v)
t = valuecrossing(x,y,v,opt)

Description

t = valuecrossing(x,y,v) returns the V-crossings of the function (x,y). valuecrossing uses linear interpolation. t = valuecrossing(x,y,v,opt) determines upwards, downwards or all value-crossings using the options 'up', 'down' or 'both', respectively.
