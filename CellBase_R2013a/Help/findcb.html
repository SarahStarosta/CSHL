findcb

Find CellBase.

Syntax

i = findcb(cb_name)

Description

i = findcb(cb_name) is a helper function. It finds CellBase with name cb_name in the list of initialized instances of CellBase and returns its index. It forces to choose a valid name or returns with an error.

See Also
