solo2trialevents4_auditory_gonogo

Create trial events structure.

Syntax

solo2trialevents4_auditory_gonogo(sesspath)
solo2trialevents4_auditory_gonogo(sesspath,ifsave)

Description

solo2trialevents4_auditory_gonogo(sesspath) creates trial events structure ('TrialEvents') for the session given sesspath. The event are extracted from the saved behavior file. The first trial is omitted. Most of the events are given relative to the TrialStart event.

solo2trialevents4_auditory_gonogo(sesspath,ifsave) uses an addidional logical input argument do determine whether to save the results.
