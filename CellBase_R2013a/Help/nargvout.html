nargvout

Number of output arguments.

Syntax

n = nargvout(funhandle,funargs)

Description

n = nargvout(funhandle,funargs) returns the number of output arguments for the function funhandle by calling it with funargs (or the first cell in cellidlist if funargs is empty). Note that nargout is not a valid alternative because here the length of output is determined.

See Also
