tetrodepairs

Find cells from the same tetrode in a session.

Syntax

[number,tetrodepartners] = tetrodepairs(cellid)
[number,tetrodepartners] = tetrodepairs(cellid,includeself)

Description

[number,tetrodepartners] = tetrodepairs(cellid) returns the total number of units on the same tetrode cellid as well as the list of cell IDs from the same tetrode.

tetrodepairs(cellid,includeself) accepts a logical input argument controling whether the unit for which the function was called is included in the number and the list.
