getvalue

Extract all values associated with a property.

Syntax

v = getvalue(prop)
v = getvalue(prop,cellids)

Description

v = getvalue(prop) returns a vector of values of the specified property (prop) for all cellids in CellBase .

v = getvalue(prop,cellids) returns a vector of values of the specified property (prop) for each cell in cellids.
