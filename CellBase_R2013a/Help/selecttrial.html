selecttrial

Execute logical expression to filter trials.

Syntax

trials = selecttrial(te,selstr)
valid_trials = selecttrial(te,'hit==1|falsealarm==1');

Description

trials = selecttrial(te,selstr) returns the trial indices (for trial structure te) satisfying the required criteria set up by selstr, which should be a logical expression.

Example

valid_trials = selecttrial(TE,'Hit==1|FalseAlarm==1');
