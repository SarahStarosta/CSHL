smoothed_psth

Smoothed PSTH.

Syntax

[spsth, conv_psth] = smoothed_psth(psth,dt,sigma)

Description

[spsth, conv_psth] = smoothed_psth(psth,dt,sigma) calculates smoothed PSTH (spsth) from psth at time resolution dt. sigma determines the smoothing Kernel (3*sigma wide). Filtered PSTH without normalization is returned in conv_psth.
