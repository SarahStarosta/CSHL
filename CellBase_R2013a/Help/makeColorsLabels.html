makeColorsLabels

Colors, labels and line style.

Syntax

[mylabels, mycolors, mycolors2, mylinestyle] = makeColorsLabels(fun_definitions,tags)

Description

[mylabels, mycolors, mycolors2, mylinestyle] = makeColorsLabels(fun_definitions,tags) defines colors, labels and line style for plotting in CellBase using the colors defined by fun_definitions.

See Also
