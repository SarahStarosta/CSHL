plot_raster_psth

Plot raster plots and PSTHs.

Syntax

hs = plot_raster_psth(cellid)

Description

hs = plot_raster_psth(cellid) plots raster plots and PSTHs aligend to 'BurstOn' and/or 'PulseOn' events. Figure handles are returned in hs. Which rasters/PSTHs are plotted is controlled by the following optional input argument parameter-value pairs (with default values):

'BurstOn', true - if true, plot 'BurstOn' raster and PSTH

'PulseOn', false - if true, plot 'PulseOn' raster and PSTH

.

Output (fields of hs struct):

H_BurstOn - figure handle for 'BurstOn' raster and PSTH

H_PulseOn - figure handle for 'PulseOn' raster and PSTH

See Also

viewcell2b, plot_raster2a
