nontetrodepairs

Find cells from different tetrodes in a session.

Syntax

[number,nontetrodepartners] = nontetrodepairs(cellid)

Description

[number,nontetrodepartners] = nontetrodepairs(cellid) returns the total number of units on other tetrodes as well as the list of cell IDs from other tetrodes.

See Also
