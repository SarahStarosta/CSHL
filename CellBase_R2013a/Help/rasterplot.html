rasterplot

Raster plot.

Syntax

h = rasterplot(binraster)
h = rasterplot(binraster,time)

Description

h = rasterplot(binraster) plots spike raster from binraster. It shows a 'histogram' view if spikes are overlapping on the plot (more spikes appear in darker colors). Zoom is implemented as buttondown function (usage is the same as the usual zoom).

h = rasterplot(binraster,time) uses the time array for specifying x-coordinates.

rasterplot(binraster,time,h) and rasterplot(binraster,[],h) plots in the figure with handle h.
