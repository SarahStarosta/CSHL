loadcb

Load the CellBase files.

Syntax

loadcb
out = loadcb(cellid,filetype)

Description

loadcb loads CellBase (CELLIDLIST, ANALYSES, TheMatrix) into the caller function's workspace.

out = loadcb(cellid,filetype) loads the specified file (filetype) of a given cell (cellid) to out, where filetype may be 'TrialEvents', 'StimEvents', 'STIMSPIKES', 'EVENTSPIKES', 'Continuous' or 'Waveforms' (see also cellid2fnames).

See Also

initcb, cellid2fnames
