addnewsessions

Adds new sessions to CellBase.

Syntax

ok = addnewsessions

Description

ok = addnewsessions adds new sessions to CellBase by creating TrialEvent files (if they do not exist). addnewsessions returns 1 if sessions were added successfully and 0 otherwise. It calls findallcells to extract new cells and session from CellBase default directory.<\p>

addnewsessions('force') creates new TrialEvents files even if they already exist.

See Also

addnewcells, findallcells
